About Me


I love learning new things. I have a strong curiosity on how things work and what makes them tick underneath.

I have always had this passion for learning since I can remember. I used to read the encyclopedia as a young child going through each volume diligently. Even up to this day, I can be found constantly reading or researching. I have a fondness for science and facts and what is true.

That curiosity and passion for knowledge is what brought me into programming and specifically web development. I love the idea that I can just create beautiful things with a computer and what seems like an unlimited selection of technologies, and be able to find solutions and answers with a simple search on the internet. I had some formal programming training and I am self-taught as a web developer, I have learned everything I know about web development through the internet, be it Udemy courses or youtube channels, from W3 schools, Stackoverflow, MDN and many more valuable sites.

My experience as an aircraft technician in the airforce and my desire to tinker and fix the car and the plumbing or electrical issues in the house, all stem from the same inner desire of finding solutions to problems and having that feeling of satisfaction.

The same capabilities and attitude carries well into web development, with its challenges of finding the exact solutions. I thrive on challenges.

When not coding, I enjoy playing hockey and going out for walks and enjoying the beatiful outdoors.

I am down to earth, easy to talk to and I get along really well with everybody. I appreciate you taking the time to find out who I am and what I do.

Portfolio

Quinfinite Scroll

Code

This web app scrolls through random images infinitely.

It uses the Unsplash API to grab the random images....

It initially loads 5 images and as you scroll down and nears the bottom of the scrollbar, an event is triggered and it will load an additional 30 images.

Again this process can continue on indefinitely.

Nations

Code

This web app provides some information on the countries of the world.

It uses the REST Countries API to grab the flag image and pertinent demographic information....

It has a dropdown selector for loading different regions of the world.

It also has a search capability that scrolls up or down to an individual country.

An option for light or dark theme is available.

React was used to code this app.

IP Tracker

Code

An app that will identify the location of an IP or a valid URL address on the map.

It uses the ip-api API to grab the IP's location information....

The IP address, location, timezone and the ISP name is shown.

The OSM API is used to grab and pinpoint the latitude and longitude on the OSM map.
